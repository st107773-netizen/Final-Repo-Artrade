# SLA & Buyout Intelligence

Аналитико-инженерное решение для управления SLA, прогнозирования выкупа и снижения потерь выручки.

## Что мы сделали
1. Подготовили и очистили данные.
2. Провели когортный анализ пути заказа до выкупа.
3. Выявили факторы, влияющие на выкуп.
4. Обучили ML-модель для раннего выделения high-risk заказов.
5. Оценили финансовые потери и потенциальный эффект улучшений.
6. Сформировали и приоритизировали управленческие рекомендации.
7. Упаковали решение как аналитический pipeline в Jupyter и Docker/API сервис.

## Ключевые результаты
- После очистки для когортного анализа осталось 7067 валидных сделок.
- Медианный итоговый выкуп по когортам: 96.4%.
- Медианный срок до выкупа: 7 дней.
- Главный фактор: `speed_to_delivery_days`.
- Лучшая модель: GradientBoosting, ROC-AUC 0.763, PR-AUC 0.946.
- Общий объём оценённых потерь: 12.41 млн ₽.
- Потенциал только от ускорения передачи в доставку: ~80 заказов и ~0.94 млн ₽.
- Топ-приоритет: аудит и оптимизация delivery service `СДЭК до ПВЗ`, ожидаемый эффект ~1.19 млн ₽.
